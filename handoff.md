# Handoff — Life Calendar

## Current Status
- **Phase:** Design / Brainstorming
- **Design decisions made:** See `docs/superpowers/specs/` for full spec (in progress)

## Decisions Made
1. **Target:** Free-form input — anyone (parents, pets, partner, self)
2. **Grid units:** Day/week/month toggle
3. **Layout:** Dashboard with cards → detail grid page
4. **Storage:** localStorage + base64 URL sharing
5. **Stack:** Next.js + Tailwind + next-intl + Vercel
6. **Language:** English default, i18n ready for multi-language
7. **Tone:** Warm & emotional — pastel, rounded, soft

## Features Planned
- Life grid visualization (filled/empty squares)
- Daily reminder ("234 Saturdays left with Mom")
- Season counter ("8 summers left with Dad")
- Time ratio bar (time spent vs time remaining)
- Meeting frequency calculator (remaining weekends together)
- URL sharing for family
